# Student Management - React Frontend

This is a React + Vite frontend that talks to the existing backend at `http://localhost:8080/api/students`.

Quick start:

1. Install dependencies:

```bash
cd frontend-react
npm install
```

2. Run dev server:

```bash
npm run dev
```

The app will be available at the URL printed by Vite (default http://localhost:5173).

Notes:
- The backend API is unchanged; the client calls the same endpoints.
- Styling uses Bootstrap 5.
